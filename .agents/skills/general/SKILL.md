# Rules
1. When solving problems related to database query statements or configurations,
  evaluate options based on both data retrieval speed and the number of table
  references. Select and present the most rational and efficient solution to
  the user.
2. When solving programming or algorithm-related problems, evaluate both
  **time complexity** and **space complexity** to select and present the most
  rational solution to the user.
3. Provide sources if they exist.
